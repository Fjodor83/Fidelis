# SUNS Logo

Inserire qui il logo SUNS.

## Formato richiesto:
- Nome file: `suns-logo.png`
- Dimensioni consigliate: 120x120 px o multipli
- Formato: PNG con sfondo trasparente

## Posizione:
Questo file deve essere salvato in:
`Fidelity.Client/wwwroot/images/suns-logo.png`

## Utilizzo:
Il logo viene visualizzato nella pagina di benvenuto responsabili al posto dell'emoji sole.
